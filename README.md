# Annyletter

**Annyletter** é um projeto pessoal antigo, criado ainda na época da escola, com o objetivo de surpreender uma pessoa muito especial.

A proposta foi desenvolver uma **cartinha digital interativa**, utilizando tecnologias básicas da web para criar uma experiência simples, mas significativa.

Hoje, o projeto permanece como um registro da minha evolução — tanto técnica quanto criativa.

---

## Sobre o Projeto

A página foi construída com **HTML, CSS e JavaScript**, explorando animações e interações para tornar a mensagem mais envolvente.

Mesmo sendo um projeto simples e experimental, ele representa uma fase importante de aprendizado e prática.

---

## Funcionalidades

* **Efeito de digitação animada**
  Simula a escrita da mensagem em tempo real

* **Interações com botões**
  Inclui elementos dinâmicos, como botão que evita o cursor

* **Mudança de tema**
  Alteração de cores baseada na proposta da página

* **Mensagens animadas**
  Pequenas animações que reforçam o tom emocional da experiência

---

## Observação

Este não é um projeto profissional — foi desenvolvido como forma de aprendizado e expressão pessoal.
